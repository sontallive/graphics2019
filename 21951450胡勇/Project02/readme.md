# 简易太阳系展示

***
+ 开发环境：macOS 10.15.2 Xcode11
+ OpenGl框架：glfw3.3 Windows pre-compiled binaries x64
 - glew2.1.0 Windows 64-bit binaries
+ 矩阵计算：glm0.9.8
+ 模型及贴图编辑：AutoDesk 3DS Max 2020
+ 模型载入：assimp 5.0.0 source
+ 开发环境：Windows10 + Visual Studio 2019 Community
+ 参考：[GLFW Documentation](https://www.glfw.org/documentation.html)
 -     [learnOpenGL](https://learnopengl.com/)
 
***
## 完成内容

+ 完成以太阳为点光源的光照系统
+ 对三个星球进行了纹理贴图
+ 使用顶点着色器和片段着色器实现了自定义光照
+ 鼠标移动控制视角，wasd控制方向
