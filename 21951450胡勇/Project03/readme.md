# 复杂三维模型展示

***
+ OpenGl框架：glfw3.3 Windows pre-compiled binaries x64
 - glew2.1.0 Windows 64-bit binaries
+ 矩阵计算：glm0.9.8
+ 模型及贴图编辑：AutoDesk 3DS Max 2020
+ 模型载入：assimp 5.0.0 source
+ 开发环境：Windows10 + Visual Studio 2019 Community
+ 参考：[GLFW Documentation](https://www.glfw.org/documentation.html) [learnOpenGL](https://learnopengl.com/)
+ 头文件及库文件在opengl.zip中
 
***
## 完成内容

+ 完成对带贴图复杂模型的载入（3ds/obj）
+ 自行实现shader多光源照射，4个光源
+ 支持键鼠操作移动视角
