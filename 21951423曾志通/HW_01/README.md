# 编程作业1：OpenGL基本程序
## 要求
* 使用glfw和glew搭建OpenGL基本框架
  * 也可以使用glut或者Qt等其他框架
* 实现基本模型绘制
  * 绘制立方体、圆球或者茶壶
* 实现相机控制
  * 支持从不同视角和距离观察模型
## 环境
* Windows + Visual Studio
* OpenGL 3.3 + GLFW
## 完成情况
* 使用glfw搭建了OpenGL基本框架
* 实现了立方体的绘制
* 实现了相机控制（WSAD移动相机，鼠标滚轮调FOV）
## 参考
* [Learn OpenGL](http://learnopengl.com/)
