# 编程作业1：OpenGL基本程序

### 1. 要求

- 使用glfw和glew搭建OpenGL基本框架
  - 也可以使用glut或者Qt等其他框架
- 实现基本模型绘制
  - 绘制立方体、圆球或者茶壶
- 实现相机控制
  - 支持从不同视角和距离观察模型

### 2. 我的环境

- win10 + clion
- OpenGL 3.3
- glfw + glad + glm

### 3. 完成情况

- 完成OpenGL基本框架
- 绘制立方体、球体
- 载入纹理
- 实现相机控制
