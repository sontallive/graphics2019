完成了第三次作业      21951086黄冠积
完成情况：
1. 使用Assimp加载了obj模型（一个外星人）
2. 在正前方添加了绿色的点光源，
   在左方添加了红色的平行光，
   在后方添加了蓝色的类似手电筒的聚光
3. 增加了鼠标滑轮的控制放大缩小功能
4. 增加了以模型为中心的旋转以及平移

控制方式：
摄像头：WASD控制前、左、后、右移动，Q控制向下移动，E控制向上移动
外星人(obj模型): 方向键控制模型的前后左右移动，J、K键控制模型的上下移动
                 鼠标滑轮控制模型的放大缩小，长按鼠标左键，模型逆时针旋转；长按鼠标右键，模型顺时针旋转
