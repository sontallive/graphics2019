# motion simulation of the sun, earth and moon

## run & interaction

Enter the Project02 folder in the terminal.

Because the CDN is used in the code to load jquery and three, network is required.

Type the following command.

`python -m http.server`

By default, access http://0.0.0.0:8000 in the browser.

Hover the mouse over a planet and its name will be displayed.
